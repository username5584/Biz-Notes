# Biz-notes

Приложение для заметок

## Требования

- Node.js 18+
- Expo Go (24.0.0) 

## Быстрый старт

```bash
# 1. Распакуйте архив и перейдите в папку проекта
cd Biz-notes

# 2. Установите зависимости
npm install

# 3. Запустите проект
npm run dev
```

## Скрипты

| Скрипт | Описание |
|--------|----------|
| `npm run dev` | Запуск Expo dev server в локальной сети|
| `npm run dev:tunnel` | Запуск Expo dev server через ngrok туннель (доступен в интернете)|
| `npm run clean` | Очистка кэша |
